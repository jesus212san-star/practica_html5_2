Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

HTML5 es esencial en la creación de páginas web, y sus elementos básicos (, , y ) permiten estructurar correctamente un documento, asegurando que el contenido sea interpretado y mostrado de manera adecuada en cualquier navegador.

Etiquetas principales

h1 - h6

En HTML5, los títulos se representan con las etiquetas h1 a h6 y sirven para estructurar el contenido de una página web de forma ordenada. Cada nivel indica la importancia del texto: h1 se usa para el título principal, mientras que h2, h3 y así sucesivamente se utilizan para dividir la información en secciones y subsecciones

p

En HTML5, los párrafos se crean con la etiqueta p, la cual se utiliza para agrupar bloques de texto dentro de una página web. Cada vez que se usa p, el navegador muestra el contenido en una nueva línea con un pequeño espacio entre párrafos, lo que mejora la legibilidad del texto.

img

En HTML5, las imágenes se insertan utilizando la etiqueta img, la cual permite mostrar gráficos dentro de una página web. Esta etiqueta es especial porque no tiene cierre y funciona mediante atributos. El más importante es src, que indica la ruta de la imagen, y alt, que proporciona una descripción del contenido (muy importante para la accesibilidad y en caso de que la imagen no cargue).

a

En HTML5, los enlaces se crean con la etiqueta (anchor), y se utilizan para conectar una página con otra o dirigir a diferentes secciones dentro del mismo sitio. El atributo más importante es href, que indica la dirección (URL) a la que se desea ir. El texto que se coloca entre y es lo que el usuario verá y podrá hacer clic.

    • HTML5 semántico

    header

    En HTML5 semántico, el encabezado se define con la etiqueta "header". Este elemento representa la parte superior de una página web o de una sección, y se utiliza para contener información introductoria o de navegación

    nav

    En HTML5 semántico, la navegación se define con la etiqueta "nav". Este elemento se utiliza para agrupar los enlaces principales que permiten al usuario moverse dentro de una página web o entre diferentes secciones del sitio

    main

    En HTML5 semántico, el contenido principal de una página se define con la etiqueta "main". Este elemento representa la parte más importante del documento, donde se encuentra la información central que el usuario busca

    section

    En HTML5 semántico, la sección se define con la etiqueta "section". Este elemento se utiliza para agrupar contenido relacionado dentro de una página, dividiéndola en partes organizadas

    article

    En HTML5 semántico, los artículos se representan con la etiqueta "article". Este elemento se utiliza para contenido independiente que puede entenderse por sí solo, como noticias, publicaciones de blog, comentarios o entradas.

    footer

    En HTML5 semántico, el pie de página se define con la etiqueta "footer". Este elemento se utiliza para representar la parte final de una página web o de una sección.

    Formularios en HTML5

    Los formularios en HTML son esenciales para la interacción entre el usuario y la web, ya que permiten capturar datos de forma organizada. Gracias a etiquetas como "form", "input" y "button", se pueden crear interfaces dinámicas y funcionales para distintos usos.

    Multimedia en HTML5

    HTML5 introdujo nuevas etiquetas que permiten insertar contenido multimedia directamente en las páginas web, sin necesidad de plugins externos como Flash. Esto facilita crear sitios más modernos, interactivos y compatibles con diferentes dispositivos

    Imágenes

    La etiqueta "img" en HTML se utiliza para insertar imágenes en una página web. Es una etiqueta vacía, lo que significa que no tiene etiqueta de cierre. Permite mostrar fotos, ilustraciones, logos y cualquier tipo de imagen para hacer el sitio más visual y atractivo

    Audio

    La etiqueta "audio" se utiliza para reproducir archivos de sonido directamente en una página web, sin necesidad de programas externos. Fue introducida en HTML5 para facilitar la integración de contenido multimedia.

    Video

    La etiqueta "video" se utiliza para insertar y reproducir videos directamente en una página web, sin necesidad de plugins externos. Fue introducida en HTML5 para facilitar el uso de contenido multimedia.

    Accesibilidad

    El atributo alt en HTML se utiliza dentro de la etiqueta para proporcionar un texto alternativo que describe la imagen. Su importancia radica en varios aspectos: primero, mejora la accesibilidad, ya que permite a las personas con discapacidad visual entender el contenido mediante lectores de pantalla. Además, es útil cuando la imagen no se carga correctamente, mostrando el texto en su lugar. También ayuda al posicionamiento en buscadores (SEO), ya que los motores de búsqueda utilizan este texto para comprender el contenido de la imagen. Por estas razones, el atributo alt es esencial para crear páginas web más inclusivas y funcionales.

    Conclusión

    HTML5 es una herramienta fundamental en el desarrollo web actual, ya que permite crear páginas más estructuradas, accesibles y dinámicas. Gracias a sus nuevas etiquetas semánticas, facilita la organización del contenido y mejora la experiencia del usuario. Además, incorpora funciones como formularios avanzados y soporte multimedia sin necesidad de plugins externos. En conclusión, HTML5 es la base para construir sitios web modernos, eficientes y adaptados a las necesidades actuales de la tecnología.
